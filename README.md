# GitQuest — учим git играя 🎮

Браузерный тренажёр: вводишь настоящие git-команды и сразу видишь, как меняется дерево коммитов. 19 уровней в 5 мирах — от первого коммита до rebase, конфликтов и работы с удалёнкой.

## Запуск

```bash
cd git-quest
npm install        # один раз
npm run dev        # откроется на http://localhost:5173
```

## Команды

| Команда | Что делает |
|---------|-----------|
| `npm run dev` | Запустить игру (режим разработки) |
| `npm run build` | Сборка для продакшена + проверка типов |
| `npm test` | Прогнать все тесты (движок + уровни) |

## Что внутри

- **Движок git** (`src/engine`) — своя упрощённая модель: коммиты, ветки, HEAD, удалёнка. Каждая команда — чистая функция, покрыта тестами.
- **Уровни** (`src/levels`) — данные: цель, стартовое состояние, проверка, подсказка. Новый уровень = новый объект в массиве.
- **Граф** (`src/graph` + `GraphView`) — раскладка дерева и отрисовка на SVG.
- **UI** (`src/components`) — карта миров, терминал, панель уровня. Состояние — Zustand, прогресс — localStorage.

## Как играть

1. Слева — карта уровней. Справа сверху — дерево коммитов, снизу — терминал и цель.
2. Читаешь цель, вводишь git-команды в терминал, смотришь, как растёт дерево.
3. Достиг цели — загорается «✅ Уровень пройден», жмёшь «Следующий уровень».
4. Застрял — кнопки «💡 Подсказка» и «Решение». Прогресс сохраняется автоматически.

Поддерживаются: `commit`, `add`, `branch`, `switch`/`checkout`, `merge`, `log`, `status`,
`reset`, `revert`, `rebase`, `cherry-pick`, `clone`, `fetch`, `pull`, `push`,
ссылки вида `HEAD~2`, и служебные `clear` / `solution`.
